# Sound assets

Drop files here using these exact names. Missing files are silent (no errors).

| File | Role |
|------|------|
| `bgm-login.mp3` | Login screen music (also falls back to `/login.mp3`) |
| `bgm-home.mp3` | Conference / committee Home Menu music |
| `bgm-workspace.mp3` | Active panel / workspace music |
| `sfx-select.wav` | Cube / menu select |
| `sfx-confirm.wav` | Confirm / create success |
| `sfx-back.wav` | Back navigation |
| `sfx-error.wav` | Error toast |

Timer alarm remains at `/alarm_sound.wav` (project `public/` root).

Mute and volumes are controlled from the Audio (A) dock button.
